# NodeApp

Install dependencies with:
```sh
npm install
```

Initialize the database with: 
```
npm run initDB
```

Strart in development mode:
```sh
npm run dev
```

## General info

Aplication created with: 
```
npx express-generator nodeapp --ejs
```
